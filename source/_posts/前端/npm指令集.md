### 安装依赖

1. 全局安装

   ```bash
   $ npm install *** -g
   # or
   $ npm i ** -g
   ```

2. 项目安装

   ```bash
   # 保存到package.json
   $ npm install ** --save
   # or
   $ npm install ** -S
   # or
   $ npm i ** --save
   # or
   $ npm i ** -S
   
   # 不保存到package.json
   $ npm install **
   # or
   $ npm i **
   ```
