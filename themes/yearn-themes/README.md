## Setup

```
$ git clone https://github.com/Youthink/hexo-themes-yearn.git themes/yearn
```

## Update

```
cd themes/yearn
git pull
```

## Development

```bash
cd themes/yearn
```

Installation depends on and builds development environment
```bash
yarn install

yarn dev
```
or

```
npm install

npm dev
```
Modify the file in `source-src`

build

```
yarn dist
```
or

```
npm dist
```
