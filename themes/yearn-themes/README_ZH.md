## 安装

```
$ git clone https://github.com/Youthink/hexo-themes-yearn.git themes/yearn
```

## 更新

```
cd themes/yearn
git pull
```

## 配置

修改主题内的 `_config.yml` 文件

### 主题配置教程

[wiki](https://github.com/Youthink/hexo-themes-yearn/wiki)

## 开发

进入主题目录

```bash
cd themes/yearn
```

安装依赖并构建开发环境
```bash
yarn install

yarn dev
```

或者

```
npm install

npm dev

```

修改 `source-src` 内的 `js` 和 `css`

重新打包文件

```
yarn dist
```

或者

```
npm dist
```
